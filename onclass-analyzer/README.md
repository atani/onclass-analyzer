# オンクラス感想分析ダッシュボード

オンクラスの感想データを可視化・分析するWebアプリです。

## デモ

**https://atani.github.io/onclass-analyzer/**

## 特徴

- **完全ローカル処理** - データはブラウザ内でのみ処理され、サーバーにアップロードされません
- **感情分析** - ポジティブ/ネガティブをキーワードベースで自動判定
- **多角的な分析** - 先生別、ブロック別、ユーザー別など様々な切り口で分析
- **PDF出力** - 分析結果をPDFで保存可能

## 使い方

1. [ダッシュボード](https://atani.github.io/onclass-analyzer/)にアクセス
2. JSONファイルをドラッグ&ドロップ
3. 自動的に分析結果が表示されます

> **Note**: 感想データのJSONファイルは [onclass-enhancer](https://github.com/atani/onclass-enhancer) を使って取得できます。

## 分析機能

| 機能 | 説明 |
|------|------|
| 基本統計 | 総感想数、ユニークユーザー数、平均感想長 |
| 感情分析 | ポジティブ/ニュートラル/改善要望の割合 |
| 感情スコア | ポジティブ率 − 改善要望率 |
| 改善要望サマリー | キーワード別・カテゴリー別の改善要望 |
| 先生別ランキング | 感想数/ポジティブ数/改善要望数でソート可能 |
| 人気ブロック | ブロック別の感想数ランキング |
| 頻出キーワード | よく使われるキーワードの可視化 |
| アクティブユーザー | 感想投稿数が多いユーザー一覧 |
| 継続率分析 | リピーター率、ヘビーユーザー数 |

## コースフィルター

以下のコースでフィルタリング可能：

- すべて
- JFYA
- オンラインサロン
- SC養成講座
- FC養成講座
- PRE養成講座

## 技術スタック

- HTML / CSS / JavaScript（フレームワークなし）
- [Chart.js](https://www.chartjs.org/) - グラフ描画
- [html2canvas](https://html2canvas.hertzen.com/) - PDF出力用
- [jsPDF](https://github.com/parallax/jsPDF) - PDF生成

## プライバシー

- JSONデータはユーザーのブラウザ内でのみ処理されます
- 外部サーバーへのデータ送信は一切ありません
- ページをリロードするとデータは消去されます
